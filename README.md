# 🍺 飲酒カウンター v2

**今夜の飲みIQ、測ってみる？**
飲み方を見える化する、行動変容型の飲酒セルフモニタリングアプリです。
ブラウザだけで動く単一ページアプリ（HTML + JS）。インストール不要・登録不要・完全無料。

## 特徴

- **かんたん記録** — ビール・ウイスキー・焼酎などカテゴリから、サイズ・飲み方・濃さを選んで記録。純アルコール量(g)を自動換算
- **シラフ予測時刻** — 1杯追加するたびに予測が後ろへずれるのが見える。これが一番のブレーキ
- **血中アルコール濃度グラフ** — 推定血中アルコール濃度の実績（実線）と予測（点線）、ピーク値をリアルタイム描画
- **飲みIQ** — 適応型の計算テストで認知機能を自己測定。シラフの自分＝120としてスコア化
- **今夜の成績表** — 上限・ペース・水分・認知機能の5バッジでS〜D判定＋称号
- **翌朝の振り返り** — 二日酔い度・後悔度・症状・対処を記録し、「自分の失敗ライン」を推定
- **安全ガイド** — 気分が悪いときはユーモアを止めて、安全行動の優先順位を固定表示

## 使い方

1. GitHub Pages の URL をスマホで開く
2. （推奨）ホーム画面に追加: Safariなら共有ボタン →「ホーム画面に追加」
3. 設定タブで体重・性別・今夜の上限を入力
4. **シラフのうちに基準テストを実施**（飲みIQに必要）
5. 飲むたびにワンタップで記録。水を飲んだら💧も忘れずに

## プライバシー

すべてのデータは**お使いの端末のブラウザ内（localStorage）のみ**に保存されます。
外部へのデータ送信は一切ありません。サーバーもアカウントも存在しません。

## ⚠️ 免責事項

- 血中アルコール濃度（BAC）とシラフ予測は **Widmark式に基づく粗い推定**です。
  体調・空腹・加齢・服薬などにより、実際の酔いはこれより強く出ることがあります
- 「飲みIQ」および認知チェックは**医学的な知能検査・認知機能検査ではありません**。
  シラフ時の自分との比較によるセルフモニタリング用のお遊び指標です
- 本アプリは**運転可否の判断・医学的判断には使用できません。飲んだら乗らない。**
- 服薬・対処の記録欄は本人の記録用であり、アプリが服薬を推奨するものではありません
- 体調に不安があるときは、ためらわず医療機関・救急（119）へ
